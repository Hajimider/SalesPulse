# 销售订单数据分析

本项目基于某电子产品公司 2022–2024 年的销售订单数据，使用 Python（Pandas、Matplotlib、Seaborn）进行数据清洗、探索性分析和可视化，输出核心经营指标、产品/品类/区域表现、时间趋势及异常检测，并给出业务优化建议。

## 📊 数据集概览
- **记录数**：3,500 条订单
- **字段**：订单日期、产品名称、类别、地区、数量、销售额、利润
- **时间范围**：2022–2024 年
- **数据质量**：无缺失值、无重复行，所有订单利润均为正

## 🔧 分析流程
1. **数据清洗**  
   - 列名汉化，日期格式转换  
   - 缺失值/重复值检查（均无）
2. **核心指标计算**  
   - 总销售额、总利润、总销量、平均利润率  
   - 盈利/亏损订单统计
3. **多维度分析**  
   - 产品排行（销售额/利润 Top10）  
   - 类别表现（销售额、利润率、销量）  
   - 区域销售对比（销售额、利润、销量、利润率）  
   - 年度/月度趋势（含环比增长与异常波动标注）
4. **交叉分析**  
   - 区域 × 类别热力图 / 堆积图  
   - 销量与利润相关性（散点图）  
   - 每单利润率分布（箱线图）
5. **异常检测**  
   - 月度销售额环比增长率，标记超过 ±50% 的异常月份

## 📈 关键结果摘要
| 指标 | 数值 |
|------|------|
| 总销售额 | 1,066.8 万元 |
| 总利润 | 184.5 万元 |
| 平均利润率 | 17.29% |
| 总销量 | 17,261 件 |
| 盈利订单 | 3,500（100%） |

- **明星产品**：Camera、Monitor、Printer 贡献最高销售额和利润  
- **强势品类**：Electronics（占销售额 50%），Accessories 利润率最高（17.3%）  
- **领先区域**：西部地区销售额和利润均居首位  
- **时间特征**：2023 年业绩峰值，2024 年略有回落；2024 年 3 月出现异常高增长  
- **相关性**：销量、均价与利润呈显著正相关

## 💡 业务建议（摘要）

- **产品策略**：强化 Camera/Monitor 组合，深耕 Keyboard/Laptop 高端线，优化长尾 SKU
- **品类调整**：资源倾斜 Electronics 和 Accessories，重塑 Office 品类（捆绑销售/精简）
- **区域布局**：西部设运营中心，北部开展专项促销，差异化推广
- **时间节奏**：管控利润率下滑，复盘 2024 年 3 月异常增长，复制成功经验
- **客户价值**：提升客单价（满减/第二件折扣），建立 VIP 体系，交叉销售

## 📝 总结

公司经营健康（全订单盈利、核心产品明确），但存在品类不均衡、区域差异、利润率微降等问题。建议聚焦 **“做强核心品类、突破弱势区域、管控利润率下滑”** 三大方向，预期可实现下一年度 **15%–20%** 业绩增长。
